# CONCEPT.md — サイトの思想

## このサイトは何か

単なるポートフォリオではなく、**私の記憶・思考・作品・活動の点に、誰かが偶然触れるための空間**です。

## 思想の核

日々の記憶、感情、違和感、風景、言葉、作品は、時系列に整列するものではありません。
それらは空間に漂う点のようなもので、見る人がそこに触れたとき、初めて意味を持ちます。

このサイトは、その「触れる体験」を作るための場所です。

## 6つの点

| カテゴリ | 意味 |
|---------|------|
| 言葉 | 本・歌・映画・漫画など、自分の中に残ってしまった一節 |
| 歩いてみる | 光・風・海・街。いいなと思った場所へ歩いた記録（写真） |
| 遊び | 役に立つかわからないが、思いついたから作ったもの（アプリ・実験） |
| note | 考えていたことが少しだけ長い文章になった場所 |
| 余白 | 家でも職場でも学校でもない、誰かと何かが起きるための余白（場づくり・活動） |
| 凪 | 見ているあいだ、心が少し凪いでくれたらという動画たち |

## 体験の設計

1. 空間に点が漂っている
2. 6つの言葉が、その空間に浮かんでいる
3. 言葉に触れると、周囲の点が集まる
4. クリックすると、点が枠を形づくり、詳細が現れる
5. 閉じると、点がまた散って空間に戻る

## このサイトが目指さないもの

- 実績を並べたポートフォリオ
- 分かりやすく整理されたリンク集
- 情報量で圧倒するサイト

## このサイトが目指すもの

- 偶然の出会いが起きる空間
- 点と点が、見た人の中で意味を結ぶ体験
- 誰かがここに来て、少し静かな気持ちになれる場所
