# DESIGN.md — デザイン方針

## 印象・雰囲気

- 宇宙、海、霧、風、余白、記憶、静けさ、光の粒
- 夜の公園、瀬戸内の海を見て黄昏れる感じ
- 暗さの中にある静かな気配

## 避けたい印象

- 派手すぎる SF 風・ネオン・グロー
- 企業サイト・ポートフォリオサイトっぽさ
- 情報が詰まりすぎた画面
- 重すぎる 3D（WebGL 必須の複雑な表現）
- ただの作品一覧・実績紹介

## カラー

| 用途 | 値 | 備考 |
|------|-----|------|
| 背景 | `#060810` → `#0a0e1c` | 暗い群青〜濃紺のグラデーション |
| 点の色 | `rgba(188, 210, 240, α)` | 青白い光の粒 |
| テキスト（通常）| `rgba(255,255,255,0.44)` | 静かに浮かぶ |
| テキスト（hover）| `rgba(255,255,255,0.92)` | 触れると明るくなる |
| パネル背景 | `rgba(9,11,19,0.92)` | 暗い半透明ガラス |
| パネルボーダー | `rgba(255,255,255,0.07)` | ほとんど見えない縁 |

## タイポグラフィ

- 見出し：Noto Serif JP・weight 300・letterSpacing 0.12〜0.16em
- 本文：Noto Sans JP・weight 300・lineHeight 1.9〜2.1
- 欧文補助：Georgia, serif

## アニメーション原則

- 動きは「機械的」ではなく「引き寄せられる」感じ
- lerp factor: 0.04〜0.06（ゆっくりなめらかに）
- トランジション: 0.32〜0.4s ease
- 点が「集まる」「散る」動きを中心に据える

## コンポーネントの見た目

- 角丸は最小限（`border-radius: 1px` 程度）
- 線は細く（1px）・色は薄く
- ボタンは枠線のみ・背景なし・文字色も控えめ
- 余白を多めに取る
